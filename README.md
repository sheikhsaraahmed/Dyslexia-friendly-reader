# Dyslexia-friendly-reader

A web-based reading tool that allows users to enter text and read it in a dyslexia-friendly format, with built-in text-to-speech support to improve accessibility.

## Features
- Text input area for custom reading content
- Text-to-speech functionality for auditory reading support
- Adjustable font size and line spacing
- Clean, minimal, and distraction-free reading interface
- Designed to reduce visual stress while reading

## Technologies Used
- HTML
- CSS
- JavaScript

## Motivation
This project was developed to explore accessibility-focused web design and to assist users with dyslexia by combining readable layouts with audio-based reading support.

- Add dyslexia-optimized fonts
